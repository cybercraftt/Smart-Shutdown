 💻 Smart Shutdown v1.0 (Исходный код)
====================================================================

Привет! Это открытый исходный код утилиты Smart Shutdown. 
Используйте эту инструкцию для запуска программы напрямую из-под Python 
или проверки её безопасности.

📋 ТРЕБОВАНИЯ ДЛЯ ЗАПУСКА:
--------------------------------------------------------------------
* Операционная система: Windows 10 / 11
* Установленный Python версии 3.11 или выше.
  (При установке Python обязательно поставьте галочку "Add Python to PATH")

📂 СТРУКТУРА АРХИВА:
--------------------------------------------------------------------
* smart_shutdown.py  — главный файл скрипта с логикой программы.
* requirements.txt   — список зависимостей (библиотека psutil для отслеживания сети).
* README.txt         — эта инструкция.

🚀 ИНСТРУКЦИЯ ПО ЗАПУСКУ (ЗА 3 ШАГА):
--------------------------------------------------------------------
ШАГ 1: Распакуйте архив с исходным кодом в любую удобную папку.

ШАГ 2: Установите необходимые библиотеки.
       - Откройте командную строку (cmd) в папке с проектом.
       - Введите команду и нажмите Enter:
       
         pip install -r requirements.txt

ШАГ 3: Запустите программу.
       - Введите в консоли команду:
       
         python smart_shutdown.py

🛠 ДЛЯ РАЗРАБОТЧИКОВ (СБОРКА В .EXE):
--------------------------------------------------------------------
Если вы хотите скомпилировать этот скрипт в самостоятельный .exe файл, 
используйте библиотеку PyInstaller:

1. pip install pyinstaller
2. pyinstaller --onefile --noconsole smart_shutdown.py

Готовый экзешник появится в папке /dist/.

❌ ВОЗМОЖНЫЕ ПРОБЛЕМЫ И ИХ РЕШЕНИЕ:
--------------------------------------------------------------------
* Ошибка: "python не является внутренней или внешней командой..."
  Решение: Вы забыли поставить галочку "Add Python to PATH" при установке Python. 
  Переустановите Python и отметьте этот пункт в первом окне установщика.

* Ошибка: "pip не является внутренней или внешней командой..."
  Решение: Используйте команду "python -m pip install -r requirements.txt".
